## ğŸ¯ Quick Reference Cards

### useState Cheat Sheet

```tsx
// Basic
const [value, setValue] = useState(0);

// Update
setValue(5); // Direct
setValue((prev) => prev + 1); // Functional (better)

// Object
setUser({ ...user, name: "X" });

// Array
setItems([...items, newItem]);
```

### useEffect Cheat Sheet

```tsx
// Mount once
useEffect(() => {}, []);

// Update when dep changes
useEffect(() => {}, [dep]);

// With cleanup
useEffect(() => {
  return () => cleanup();
}, [dep]);
```

### Common Patterns

```tsx
// Conditional rendering
{condition && <Component />}
{condition ? <A /> : <B />}

// List rendering
{items.map(item => <Item key={item.id} />)}

// Event handling
<button onClick={() => handleClick()}>
```
